# 符籙便簽 v3.0

> 古风道家符纸风格桌面便签程序

![Python](https://img.shields.io/badge/Python-3.8+-blue.svg)
![Tkinter](https://img.shields.io/badge/GUI-Tkinter-green.svg)
![Platform](https://img.shields.io/badge/Platform-Windows-lightgrey.svg)

## ✨ 功能特性

- 🎨 **古风界面** - 朱砂红、宣纸色等 10 种符纸颜色（4 浅色 +6 深色）
- 📌 **桌面置顶** - 便签可固定在桌面最上层
- 🔔 **悬浮球** - 桌面右下角「吉」字悬浮球，随时唤出主面板
- 💾 **自动保存** - 书写内容自动保存，无需手动操作
- 📱 **多便签** - 支持创建多个便签，独立管理
- 🗑️ **完整卸载** - 一键卸载，无残留

## 🖼️ 界面预览

- 主面板：仿古宣纸风格，便签列表管理
- 便签窗口：符籙边框装饰，古风文字编辑
- 悬浮球：朱砂红圆形，「吉」字显示

## 📦 安装说明

### 方法一：使用安装程序（推荐）

1. 下载 `符籙便簽_Setup.exe`
2. 双击运行
3. 选择安装目录（默认：`%LOCALAPPDATA%\StickyNotes`）
4. 点击「开始安装」
5. 安装完成后桌面会生成快捷方式

### 方法二：绿色版

1. 下载 ZIP 压缩包
2. 解压到任意目录
3. 运行 `installer/setup.bat`

### 方法三：源码运行

```bash
# 需要 Python 3.8+ 和 tkinter
python launch.py
```

## 🚀 使用说明

### 创建便签

1. 点击主面板底部「書符立簽」按钮
2. 选择喜欢的符纸颜色
3. 在新窗口中输入内容

### 便签操作

| 操作 | 说明 |
|------|------|
| 移动便签 | 拖拽顶部工具栏 |
| 调整大小 | 拖拽右下角缩放手柄 |
| 更换颜色 | 点击颜色选择器 |
| 固定到桌面 | 点击 📌 图标 |
| 关闭便签 | 点击 ✕ 或按 `Ctrl+W` |
| 保存便签 | 按 `Ctrl+S`（自动保存） |

### 主面板

| 功能 | 说明 |
|------|------|
| 搜索便签 | 顶部搜索框输入关键词 |
| 颜色筛选 | 点击颜色标签过滤便签 |
| 创建便签 | 点击底部「書符立簽」按钮 |
| 隐藏面板 | 点击 ⊖ 或悬浮球 |

### 悬浮球

- **点击**：显示/隐藏主面板
- **拖拽**：移动悬浮球位置
- **自动隐藏**：鼠标离开后自动缩到屏幕边缘

## 🔧 技术栈

## 📝 数据存储

便签数据保存在：
```
%APPDATA%\StickyNotes\notes.json
```

备份此文件即可备份所有便签内容。

## 🗑️ 卸载方法

### 方法一：Windows 设置
1. 打开「设置 > 应用 > 安装的应用」
2. 找到「符籙便簽 v3.0.0」
3. 点击「卸载」

### 方法二：开始菜单
1. 打开「开始菜单 > 符籙便簽」
2. 运行「卸载符籙便簽」

### 方法三：安装目录
```bash
# 进入安装目录
uninstall.bat
```

## ❓ 常见问题

### Q: 程序无法启动？
A: 确保已安装 Python 3.8+，并检查是否缺少 tkinter 模块。

### Q: 便签数据丢失？
A: 检查 `%APPDATA%\StickyNotes\notes.json` 文件是否存在。

### Q: 如何备份便签？
A: 复制 `%APPDATA%\StickyNotes\notes.json` 文件即可。

### Q: 悬浮球不见了？
A: 移动到屏幕边缘会自动隐藏，鼠标靠近会重新显示。
